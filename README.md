## 🚀 Getting Started

To get a local copy up and running, follow these simple steps.

### ✅ Prerequisites

You will need a web browser to view the website.

### 💻 Installation

1.  Clone the repo
    ```sh
    git clone https://github.com/tushar591/Portfolio.git
    ```
2.  Open `my-app` in your web browser.
